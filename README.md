# YHPhotoSelectorDemo
一个支持多选的图片和视频选择器，轻量级纯Siwft实现     A multi - choice picture and video selector, lightweight pure Siwft implementation

## Requirements
* iOS 10.0 or later
* Xcode 10.0 or later

## <a id="Howtouse"></a>How to use
* Use CocoaPods:
- `pod "YHPhotoSelector"`
* Manual import：
- Drag All files in the `YHPhotoSelector ` folder to project
