# YHPhotoSelectorDemo
一个支持多选的图片和视频选择器，超轻量级纯Swift实现     
A multi - choice picture and video selector, super lightweight pure Siwft implementation

## Requirements
* iOS 10.0 or later

- Drag All files in the `YHPhotoSelector ` folder to project
